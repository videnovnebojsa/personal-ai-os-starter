# Scaffold - Universal Prompt

> Paste this into any AI tool to set up your Personal AI OS folder structure.

## Prompt

```
Create the following folder structure in my project:

- 00_Inbox/
- 10_Projects/
- 20_Areas/
- 30_People/
- 40_Admin/

Add an empty .gitkeep file inside each folder.

These folders follow the PARA method:
- 00_Inbox - capture everything here first
- 10_Projects - active projects with clear goals
- 20_Areas - ongoing areas of responsibility
- 30_People - notes about people you meet
- 40_Admin - goals, plans, reviews, and tasks
```
