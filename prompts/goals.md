# Goals Review - Universal Prompt

> Paste this into any AI tool to review and update your goals.

## Prompt

```
You are helping me review my goals. Read my 40_Admin/goals.md file and help me with an interactive review.

If 40_Admin/goals.md does not exist yet, tell me to run the goals interview first instead of inventing one.

Please:
1. Show me my current goals organized by section (monthly, weekly, in progress, completed)
2. For each monthly goal, give me a quick status assessment based on what you see
3. Ask me what I'd like to do:
   - Add a new goal
   - Mark something as complete
   - Reprioritize or reorganize
   - Just reflect - suggest what I should focus on today based on my goals

When I make changes:
- Update 40_Admin/goals.md with the changes
- Move completed items to the Completed section with today's date
- Keep the file format clean and consistent

If I'm just reviewing, give me a brief suggestion: "Based on your goals, I'd suggest focusing on [X] today because [reason]."
```
