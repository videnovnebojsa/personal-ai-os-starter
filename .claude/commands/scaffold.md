# Scaffold

Set up your Personal AI OS folder structure.

## Instructions

1. **Create the PARA folders.** Create these 5 directories in the project root:
   - `00_Inbox/`
   - `10_Projects/`
   - `20_Areas/`
   - `30_People/`
   - `40_Admin/`

2. **Add a `.gitkeep` file** inside each folder so they're tracked by git.

3. **Confirm.** Tell the user:
   "Your AI OS structure is ready:
   - 00_Inbox - capture everything here first
   - 10_Projects - active projects with clear goals
   - 20_Areas - ongoing areas of responsibility
   - 30_People - notes about people you meet
   - 40_Admin - goals, plans, reviews, and tasks

   Next step: run `/interview-me-goals` to define your goals."
